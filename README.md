# Weather App

iOS приложение для просмотра погоды, разработанное с использованием UIKit.

## Функциональность

- Отображение текущей погоды
- Почасовой прогноз (оставшиеся часы текущего дня)
- Прогноз на 7 дней
- Автоматическое определение местоположения
- Резервное местоположение (Москва) при отказе в доступе к геолокации
- Обработка ошибок и состояния загрузки

## Технические детали

- iOS 15.0+
- Swift 5.0
- UIKit
- Программная верстка (без Storyboard)
- Архитектура: MVC
- Асинхронные сетевые запросы с использованием async/await
- CoreLocation для работы с геолокацией

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-deser7/WeatherApp.git
```

2. Откройте проект в Xcode:
```bash
cd WeatherApp
open WeatherApp.xcodeproj
```

3. Соберите и запустите проект (⌘R)

## API

Приложение использует WeatherAPI.com для получения данных о погоде:
- Текущая погода: `/current.json`
- Прогноз: `/forecast.json`
