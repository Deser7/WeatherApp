# Weather App

iOS приложение для просмотра прогноза погоды, разработанное с использованием SwiftUI.

## Функциональность

- Просмотр прогноза погоды на 5 дней
- Поиск погоды по городам
- Отображение температуры, скорости ветра и влажности
- Поддержка русского языка
- Автоматическая конвертация единиц измерения

## Требования

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/Weather.git
```

2. Откройте проект в Xcode:
```bash
cd Weather
open Weather.xcodeproj
```

3. Соберите и запустите проект (⌘R)

## Использование

1. При запуске приложение автоматически показывает прогноз погоды для Москвы
2. Используйте поисковую строку для просмотра погоды в других городах
3. Для каждого дня отображается:
   - Дата
   - Температура
   - Описание погоды
   - Скорость ветра
   - Влажность

## API

Приложение использует WeatherAPI.com для получения данных о погоде.
